Trabalhando com listas

Listas Ordenadas

Listas ordenadas seguem uma ordem numérica. Para isso usamos a tag ol, ex:

  1. Acordar
  2. Escover os dentes
  3. Tomar café

Também pode usar letra no lugar dos números, para isso usamos o parâmetro type="A" ex:

  1. Acordar
  2. Escover os dentes
  3. Tomar café

Podemos também iniciar a lista com qualquer número, para isso também usamos o parâmetro type="1" start="4" colocando o número desejado, ex:

  1. Acordar
  2. Escover os dentes
  3. Tomar café

Pode também usar números romanos grandes e pequenos, usamos os parametros type="I" para romanos grandes, e type="i" para romanos pequenos, ex:

  1. Acordar
  2. Escover os dentes
  3. Tomar café
  1. Acordar
  2. Escover os dentes
  3. Tomar café

Listas não ordenadas

Não seguem uma ordem certa. Tipo uma lista de compras, para isso usamos a tag ul, ex:

  • Pão
  • Queijo
  • Café
  • Leite
  • Manteiga

Também podemos usar o parametro type para customizar a lista, temos o tipo de disc q é padrão, circle q deixa uma bolinha e square q deixa em quadrados, ex:

  • Pão
  • Queijo
  • Café
  • Leite
  • Manteiga
  • Pão
  • Queijo
  • Café
  • Leite
  • Manteiga
